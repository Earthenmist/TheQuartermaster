Warband Nexus - Media Assets
=============================

This folder contains custom textures and icons for the addon.

Required Files:
- icon.tga or icon.blp - Main addon icon (32x32 or 64x64 recommended)

To create an icon:
1. Create a 64x64 pixel image in PNG format
2. Convert to TGA (32-bit with alpha) or BLP format
3. Name it "icon.tga" or "icon.blp"

Note: If no custom icon is provided, the addon will use a fallback Blizzard icon.

Recommended tools for texture creation:
- GIMP (free) - Can export TGA directly
- BLP Converter - For BLP format conversion
- Photoshop with BLP plugin
